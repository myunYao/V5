# Icon 图标
## 代码示例  

> 更多完整示例地址： example/src/views/base/icon/   
## 图标使用方法    

直接放至标签中 
   
HTML 
```html  
<i>&#xe612;</i>
```  
通过class添加，例如：
     
HTML  
```html  
<v5-icon class="apple"/>
```    
class内容为图标名称  

## props 参数
| 参数 | 类型 | 说明 | 默认值 |
| --- | --- | --- | --- |
| class | `String` | 图标名称 | - |
